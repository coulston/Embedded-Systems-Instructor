# Testing Procedure for Development Boards

Periodically the development boards need to be tested to insure that students
are starting the course with properly functioning hardware.  The testing 
procedure included in this folder will help you efficiently and effectively
perform this testing.

## Supplies

In order to do the testing you must have the following supplies.  It is 
recomended to have these supplies in a box labeled "383 Dev Board Test Kit."

1. PICkit 4
2. 2GB SD Card with testing software installed
3. 3 2-pin jumpers
4. A 128MB SD Card

In addition consider adding the following items to the test kit.

1. A USB SD Card Reader to write the necessary files onto the 2GB SD Card used by the PICkit 4.
2. Small screwdriver to tighten the screw terminal used to attach the speaker
3. A small speaker to test the audio output.
